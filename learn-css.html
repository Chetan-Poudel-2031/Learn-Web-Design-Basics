CSS Basics

CSS Styling

CSS (Cascading Style Sheets) is used to style HTML elements. Here are some basic CSS properties:

  • color: blue;: Sets the text color.
  • font-size: 16px;: Sets the font size.
  • margin: 10px;: Adds space outside an element.
  • padding: 20px;: Adds space inside an element.

Example:


p {
    color: blue;
    font-size: 16px;
    margin: 10px;
    padding: 20px;
}

CSS Box Model

The CSS box model defines how elements are laid out. It consists of four parts:

  • Content: The actual content of the element.
  • Padding: Space between the content and the border.
  • Border: The edge of the element.
  • Margin: Space outside the border.

Example:


div {
    width: 200px;
    padding: 20px;
    border: 5px solid black;
    margin: 10px;
}
