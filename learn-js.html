JavaScript Basics

What is JavaScript?

JavaScript is a programming language used to add interactivity to webpages. It allows you to manipulate HTML and CSS elements, handle user interactions, and create dynamic content.

Here's a simple example of JavaScript in action:


<button onclick="alert('Hello, World!');">Click Me</button>

When the button is clicked, a pop-up message saying "Hello, World!" will appear.

JavaScript Functions

Functions are reusable blocks of code that perform a specific task. They help you organize your code and avoid repetition.

Here's how to define and call a function:


function greet() {
    alert("Hello, welcome to the website!");
}

// Call the function
greet();

You can also pass parameters to functions:


function greetUser(name) {
    alert("Hello, " + name + "!");
}

// Call the function with a parameter
greetUser("Alice");

DOM Manipulation

The DOM (Document Object Model) represents the structure of a webpage as a tree of objects. JavaScript allows you to manipulate the DOM to change the content, structure, or style of elements.

Here are some common DOM manipulation techniques:

  • Change Text: 
    
document.getElementById("myElement").innerText = "New Text";
  • Change HTML: 
    
document.getElementById("myElement").innerHTML = "<strong>New Text</strong>";
  • Change Style: 
    
document.getElementById("myElement").style.color = "red";

Event Handling

JavaScript allows you to respond to user actions, such as clicks, mouse movements, and keyboard input. This is done using event listeners.

Here's an example of adding a click event listener to a button:


<button id="myButton">Click Me</button>

<script>
    document.getElementById("myButton").addEventListener("click", function() {
        alert("Button clicked!");
    });
</script>

When the button is clicked, a pop-up message will appear.

Variables and Data Types

Variables are used to store data in JavaScript. You can declare variables using let, const, or var.

Here are some examples:


let name = "Alice"; // String
const age = 25;     // Number (constant)
var isStudent = true; // Boolean

JavaScript supports several data types, including:

  • Strings: Text data, e.g., "Hello".
  • Numbers: Numeric data, e.g., 42.
  • Booleans: True or false values.
  • Arrays: Lists of data, e.g., [1, 2, 3].
  • Objects: Key-value pairs, e.g., { name: "Alice", age: 25 }.
