HTML Basics

HTML Structure
Learn about the structure of a webpage with basic HTML tags.

HTML (HyperText Markup Language) is the standard language for creating web pages. It defines the structure and layout of a webpage using various tags. Let's break it down:

  • <!DOCTYPE html>: This declaration defines the document type and version of HTML (HTML5 in this case).
  • <html>: The root element of an HTML document. Everything else is nested inside this tag.
  • <head>: Contains meta-information about the document, such as the title, character encoding, and links to external resources like stylesheets or scripts. This section is not visible on the webpage.
  • <body>: Contains the visible content of the webpage, such as text, images, links, and other elements.

Example:


<!DOCTYPE html>
<html>
  <head>
    <title>My First Webpage</title>
  </head>
  <body>
    <h1>Welcome to My Webpage!</h1>
    <p>This is a paragraph of text.</p>
  </body>
</html>
HTML Tags & Attributes
Understand the role of attributes in HTML and how to use them to add functionality and styling.

HTML tags define the structure and content of a webpage. Attributes provide additional information about an element and are always included in the opening tag.

  • <a href="URL">: Creates a hyperlink to a specified URL using the href attribute. Example: <a href="https://www.example.com">Visit Example</a>.
  • <img src="image.jpg" alt="Description">: Displays an image using the src attribute to specify the image location and the alt attribute to provide alternative text for accessibility. Example: <img src="logo.png" alt="Company Logo">.
  • <div class="container">: Defines a division or section with a specified class for styling. Example: <div class="header">This is the header</div>.

Common Attributes:

  • class: Used to apply CSS styles to multiple elements.
  • id: Used to uniquely identify an element.
  • style: Used to apply inline CSS styles.
  • title: Provides additional information about an element (shown as a tooltip).
Common HTML Elements
Learn about the most commonly used HTML elements.

Here are some of the most commonly used HTML elements:

  • <h1> to <h6>: Headings. <h1> is the largest and <h6> is the smallest.
  • <p>: Paragraphs. Used for blocks of text.
  • <ul> and <ol>: Unordered and ordered lists. Use <li> for list items.
  • <table>: Creates a table. Use <tr> for rows, <th> for headers, and <td> for data cells.
  • <form>: Used to create forms for user input. Includes elements like <input>, <button>, and <textarea>.

Example:


<h1>This is a Heading</h1>
<p>This is a paragraph.</p>
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
